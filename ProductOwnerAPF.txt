Вводная: Онлайн магазин. Продажа товаров. Вход через авторизацию. Катологи, поиск товаров, карточки товаров. Доставка курьерской службой. 
Оплата онлайн картой. Есть сторонний сервис REST API, который хранит информацию о клиентах, размещает заказы.


Требования от Product Owner: 
"Добавить возможность оплаты APF через сайт, после его истечения"

Детали:
APF - ежегодный обязательный взнос для всех Клиентов. Без оплаченного APF клииент не может совершать заказы.

APF является электронным товаром. Нельзя смешивать в корзине с другими товарами.
APF для Клиентов с категорией SP и остальных Клиентов имеют разные SKU а именно:
 - APF для SP: SKU = 0909
 - APF для остальных категорий Клиентов: SKU = 9909

APF не должен быть доступен на сайте через каталог/поиск товаров.

Дата истечения APF будет приходить в специальном поле в REST API в GetDistributorInfo методе.
